Test ProjectHub Views

Steps to test Kanban Board:

  1. Login to ProjectHub below
  2. Click on "Board" in the navigation
  3. Select a project from the dropdown (e.g., "Web Application")
  4. You should see tasks organized in columns

Steps to test Analytics:

  1. Click on "Analytics" in the navigation
  2. You should see:
    • Stats cards (Total Projects, Total Tasks, etc.)
    • Three charts: Project Status, Task Priority, Task Timeline
