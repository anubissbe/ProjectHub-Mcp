DEPLOYMENT COMMAND FOR SYNOLOGY
===============================

Run this on your Synology NAS via SSH:

wget -O- http://172.28.173.145:8080/projecthub-complete-bundle.tar.gz | sudo tar -xz -C /volume1/docker/projecthub && cd /volume1/docker/projecthub && sudo docker-compose -f docker-compose.synology-minimal.yml up -d

Before running the above command:
1. Start HTTP server on this machine: python3 -m http.server 8080
2. Make sure /volume1/docker/projecthub exists or will be created

Alternative step-by-step:
1. On this server: cd /opt/projects/projects/projecthub-mcp-server && python3 -m http.server 8080
2. On Synology: 
   sudo mkdir -p /volume1/docker/projecthub
   cd /volume1/docker/projecthub
   sudo wget http://172.28.173.145:8080/projecthub-complete-bundle.tar.gz
   sudo tar -xzf projecthub-complete-bundle.tar.gz
   sudo docker-compose -f docker-compose.synology-minimal.yml up -d
